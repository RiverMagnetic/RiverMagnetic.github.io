Just Add Water

This single page application is a gentle personal growth app. Users can create, edit, view, and delete their own private journal entries. It is styled as a mock-up for a mobile app.

Technologies used

PostgreSQL, Ruby on Rails, JavaScript, HTML, CSS, SCSS, Bootstrap, SQL, AJAX, and JQuery

Bucket.li

Team project: this app lets users create and store bucket list items focused on travel adventures. Users name their bucket list items, describe the adventure they want to have, and choose the location where they want to have it from a drop-down menu. They can edit, update, and delete their items and view all bucket list items.

Technologies used

MongoDB, Mongo, HTML, Bootstrap, and JavaScript, using Mongoose, Express, AJAX, and JQuery

My List

This app lets a user create a list by adding items with descriptions after they have signed in. When users edit, delete, or add items, the list automatically updates. The app's resources are private to each user, so users can only perform CRUD actions on their own list items.

Technologies used

Ruby on Rails, PostgreSQL, JavaScript, Handlebars, HTML, CSS, and Bootstrap

Tic-Tac-Toe

A two-player game where players compete to mark three boxes in a row. This is a fully functional web app with game logic to determine wins and ties.

Technologies used

HTML, CSS, and JavaScript, and JQuery, and using AJAX to interact with a third-party API
